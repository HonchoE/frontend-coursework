# Features

## Core Functionality

### ✅ Lesson Browsing
- Display all available after-school classes
- Show lesson details (subject, location, price, spaces)
- Visual icons for each subject

### ✅ Search and Filter
- Real-time search across subjects and locations
- Sort by subject, location, price, or availability
- Ascending/descending order toggle

### ✅ Shopping Cart
- Add lessons to cart
- Remove lessons from cart
- Real-time space tracking
- Cart item counter

### ✅ Checkout Process
- Form validation (name: letters only, phone: numbers only)
- Order submission to backend API
- Success confirmation modal
- Automatic cart clearing

### ✅ Responsive Design
- Mobile-friendly layout
- Tablet optimization
- Desktop experience

## Technical Features

- Vue 3 Composition API
- Component-based architecture
- RESTful API integration
- Modern ES6+ JavaScript
- CSS3 animations
- Font Awesome icons
